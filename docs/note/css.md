## 相对定位，绝对定位

relative： 占位不脱标，相对于自身在原文档流中位置，不改变元素大小
absoulte: 脱标不占位，参照上级或者上上级有无定位，没有则根据 body 元素定位，改变元素大小，不设置宽高，依赖内容决定
