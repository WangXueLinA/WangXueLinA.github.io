# 高亮字体

维护人： xuelin

<!-- > 高亮字体 -->

## 安装

```sh
npm i @wxl/hight-light-title
or
yarn add @wxl/hight-light-title
```

## API

| 参数      | 说明                   | 类型                | 默认值 |
| --------- | ---------------------- | ------------------- | ------ |
| text      | 标题                   | string              | -      |
| keyWord   | 关键字                 | string              | -      |
| style     | 自定义关键字的样式     | React.CSSProperties | -      |
| className | 自定义关键字样式的类名 | string              | -      |
